- Nếu chưa có node, cài đặt nodejs với version LTS.

# 1: Cài đặt trình biên dịch

- Cài đặt trình biên dịch
  `npm i typescript`

- Biên dịch file
  `tsc filename.ts`

  # 2. Cài đặt vitejs

  - Khi muốn dùng pnpm, cài pnpm:

  `npm i pnpm`

  `pnpm create vite@latest my-name-app -- --template react-ts`

  ```
  cd my-name-app
  pnpm i
  pnpm run dev
  ```
